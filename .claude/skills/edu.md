# Edu Skill — Bullground Consulting & Education

## Descripcion
Esta skill replica el cerebro operativo de Bullground para asesorias, consultoria y educacion en herramientas de automatizacion e IA aplicada a negocios reales.

## Stack de Herramientas Bullground
- **Claude Code**: CLI de Anthropic para desarrollo asistido por IA, creacion de skills, hooks, agentes
- **GitHub**: Versionamiento, colaboracion, CI/CD, GitHub Actions
- **n8n**: Orquestacion de workflows y automatizaciones (self-hosted o cloud)
- **Apify**: Web scraping, extraccion de datos, actors para ads research
- **Vite + React**: Landing pages rapidas, funnels de conversion
- **Google Ads API / Meta Ads API**: Gestion programatica de campanas publicitarias
- **PDF Generation**: Reportes automatizados (puppeteer, jsPDF, n8n nodes)

## Casos de Uso Documentados

### 1. Landing Pages & Funnels
- Creacion de landing pages con Vite + React en minutos usando Claude Code
- Design system Bullground (cream palette, Space Grotesk, Inter)
- Formularios de captura conectados a CRM o email marketing

### 2. Automatizacion de Contenido
- Generacion de copy para ads usando Claude API
- Creacion de variantes A/B automatizadas
- Pipelines de contenido: idea → copy → imagen → publicacion

### 3. Reportes Automatizados
- n8n workflows que extraen data de Google Ads / Meta Ads API
- Generacion de PDFs con metricas clave (CTR, CPA, ROAS, conversiones)
- Entrega programada via email o almacenamiento en carpetas (Google Drive, local)

### 4. Web Scraping & Research con Apify
- Actors para extraer datos de competencia
- Scraping de ad libraries (Meta Ad Library, Google Ads Transparency)
- Enriquecimiento de leads y audiencias

### 5. Agentes & Skills con Claude Code
- Creacion de skills personalizadas (.claude/skills/)
- Hooks para automatizar procesos de desarrollo
- Agentes que combinan Claude + APIs externas

## Formato de Propuesta de Asesoria

Cuando se solicite una propuesta de consultoria, seguir esta estructura:
1. Header con branding Bullground
2. Datos del cliente y contexto
3. Alcance claro de la sesion
4. Entregables especificos
5. Inversion y forma de pago
6. Call to action
