## Bloque 1 — Operaciones & Logistica
*Automatizacion del core del negocio*

### 1. Sistema de Agendamiento Inteligente (15 min)
- Diseñar un workflow en n8n que tome las variables clave (empleado, zona, cliente, paquete, horario) y genere agendas optimizadas
- Reglas de negocio: maximo de perros por paseador, compatibilidad entre perros, zonas geograficas para minimizar traslados
- Integracion con Google Calendar o Notion para que cada empleado vea su agenda del dia automaticamente
- Alertas automaticas cuando hay cambios, cancelaciones o conflictos de horario

### 2. Gestion de Clientes & Paquetes (10 min)
- Base de datos centralizada de mascotas: dueño, paquete contratado, historial de servicios, notas de entrenamiento
- Workflow de onboarding: nuevo cliente → se crea ficha → se asigna paseador → se agenda primera sesion
- Recordatorios automaticos a clientes (WhatsApp o SMS) antes de cada sesion
- Tracking de paquetes: cuantas sesiones lleva, cuantas le faltan, renovaciones proximas

### 3. Capacitacion & Operacion del Equipo (5 min)
- Repo en GitHub con el manual de operaciones como contexto para Claude Code
- Cada empleado puede consultar protocolos de entrenamiento, manejo de emergencias y procedimientos via Claude
- Checklist automatico post-paseo: fotos, notas del perro, incidencias → se reporta al dueño

## Bloque 2 — Marketing & Crecimiento
*Contenido y captacion con IA*

### 4. Generacion de Contenido (10 min)
- Skill de Claude Code que genere contenido para redes basado en tu marca: tips de entrenamiento, historias de perritos, antes/despues
- Calendario editorial automatico: n8n genera sugerencias semanales de posts para Instagram, TikTok y Facebook
- Templates de contenido adaptados al tono del negocio (cercano, profesional, local)

### 5. Captacion & Seguimiento de Leads (10 min)
- Workflow: lead entra (formulario, WhatsApp, DM de Instagram) → se clasifica → se envia info del servicio → se agenda llamada o visita
- Bot de WhatsApp que responde preguntas frecuentes (precios, paquetes, zonas de cobertura)
- Seguimiento automatico a leads que no han respondido (a los 2 y 5 dias)

### 6. Casos de Uso por Explorar & Roadmap (10 min)
- Dashboard de KPIs: perros activos, sesiones completadas, ingresos, tasa de retencion
- Sistema de reviews automatico: despues de X sesiones, pedir reseña en Google
- Programa de referidos automatizado (cliente refiere → descuento automatico)
- Analisis de competencia local con Apify (que ofrecen, a que precios, que reviews tienen)
- Reportes semanales automaticos para ti como fundador
